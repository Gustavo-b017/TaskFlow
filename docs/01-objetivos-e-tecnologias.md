# 🎯 Objetivos e Tecnologias

O **TaskFlow** é um aplicativo de gerenciamento de tarefas pessoais. O objetivo é criar uma solução que permita ao usuário gerenciar seu dia a dia de forma eficiente e organizada.

## 🚀 O Que Você Vai Aprender?

Este projeto foi desenhado para cobrir os temas fundamentais do desenvolvimento mobile moderno:

1.  **Fundamentos do React Native:** Aprender a usar componentes como `View`, `Text`, `TextInput` e `FlatList`.
2.  **Navegação:** Criar um fluxo de telas profissional com pilhas e abas.
3.  **Persistência de Dados:** Salvar as tarefas no próprio dispositivo com `AsyncStorage`.
4.  **Consumo de API:** Buscar dados externos com `fetch` ou `axios`.
5.  **Gerenciamento de Estado:** Usar a `Context API` para gerenciar autenticação, tarefas e temas (dark/light).
6.  **TypeScript:** Aplicar tipagem forte em todo o código para evitar bugs comuns.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, escolhemos as ferramentas que são padrão no mercado:

-   **React Native:** Framework para criar aplicativos nativos usando JavaScript/TypeScript.
-   **Expo:** Ecossistema que facilita a criação e o teste do app em dispositivos reais sem complicação.
-   **TypeScript:** Adiciona tipos ao JavaScript, garantindo segurança no código.
-   **React Navigation:** Biblioteca oficial para criar navegação entre telas.
-   **AsyncStorage:** Para salvar dados localmente no celular.
-   **Fetch / Axios:** Para realizar chamadas a servidores externos.

---

## ✅ Regras de Ouro do Projeto

Para garantir um código de alta qualidade, seguimos estas regras:

1.  **Sem "any":** Todo o código deve ser tipado.
2.  **Modularização:** O código deve ser separado por responsabilidades.
3.  **Interface Limpa:** Espaçamentos e cores consistentes.
4.  **Feedback Visual:** O app deve mostrar quando algo está carregando ou se houve um erro.
